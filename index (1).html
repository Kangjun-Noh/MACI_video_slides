Multi-LLM Adaptive Conformal Inference for Reliable LLM Responses

Kangjun Noh, Seongchan Lee, Ilmun Kim, Kyungwoo Song

MACI studies how to make LLM responses more reliable by combining adaptive filtering, conformal inference, and multi-LLM signals to control risk while retaining useful responses.

Paper Code Poster / Slides Contact
Overview

Large language models are increasingly used in settings where incorrect outputs can be costly. MACI is built around a simple goal: retain helpful model outputs while filtering out unreliable ones with statistical guarantees.

Key Ideas

Adaptive filtering

Use confidence-aware rules to decide which generations should be retained or filtered.

Conformal control

Calibrate the filtering rule so the retained set satisfies a target reliability level.

Multi-LLM signals

Aggregate multiple model judgments to obtain a more stable estimate of response reliability.

Abstract

We propose MACI, a framework for improving the reliability of LLM outputs through adaptive conformal filtering. The method combines multiple LLM-based signals and a calibrated decision rule to selectively retain responses that satisfy a target factuality or correctness criterion. MACI is designed for practical use when users care about both statistical reliability and response retention.
